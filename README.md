# rpi-setup
installation d'un serveur web (paquets lamp & git) pour rpi3

# En terminal tapez:
wget https://github.com/Ko5t1C/rpi-setup/archive/master.zip ; unzip master.zip ; cd rpi-setup-master ; ./install.sh
